# vscode-setup

My personal vscode setup (python, c++)


# Extensions

```
1. https://marketplace.visualstudio.com/items?itemName=ms-python.python
   (Press Ctrl+Shift+P to select a python interpreter)
2. https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools
3. https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.activitusbar
4. https://marketplace.visualstudio.com/items?itemName=rokoroku.vscode-theme-darcula
   (Open ~/.vscode/extensions/rokoroku.vscode-theme-darcula-1.*.*/themes/darcula.json and 
    update "editorActiveLineNumber.foreground" with "#4c4c4c")
5. https://marketplace.visualstudio.com/items?itemName=wayou.vscode-todo-highlight
6. https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker
7. https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop
8. https://marketplace.visualstudio.com/items?itemName=robertohuertasm.vscode-icons
```
